# Menu Layout em WPF

## Esse é um projeto de estudo que realizei para criação de um layout simples em WPF.
<img src="10-Flat-UI\Images\print.png" alt="Banner" width="650"/>

## Tecnologias utilizadas

  - **.NET 9.0**
  - **C# 12**
  - **WPF (Windows Presentaion Foundation)**
  - **MVVM Pattern**
  - **XAML**
  - **ObservableCollection** (binding e notificação de mudanças)
  - **INotifiPropertyChanged** (atualização automática da UI)
  - **ICommand**
